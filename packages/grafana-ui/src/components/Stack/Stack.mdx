# Stack

Stack is a component for doing simple layouts, either vertically or horizontally.

### Usage

#### When to Usage

Use when you want to align multiple items in the regular flow of the website, either vertically or horizontally with tokenized spacing between them.

#### When not to use

If you need to justify content or other flex properties, use the `Flex` component instead.

If you need to handle more complex layouts, use `Grid`.
