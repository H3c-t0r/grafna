# Flex

The Flex Component aims at providing a more efficient way to lay out, align and distribute space among items in a container and
the decision to create it is to ensure consistency in design across Grafana.

### Usage

#### When to use

Use when in need to align components and small parts of the application. Use as parent container to wrap elements that
you wish to align in a certain way.

#### When not to use

When you need to lay out bigger parts of the application or when you want to create page lay out.

### Variants

Flex component has few variants that can be used based on the desire alignment you need for your case.

Some examples of how to use the Flex component can be seen below:

- AlignItems stretch

```ts
import { Flex } from '@grafana/ui'

<header>
    <h1>Using Flex component to align-items stretch and justify-content to be space-between</h1>
</header>
<Flex flexFlow="row wrap" alignItems="stretch" justifyContent="space-between" gap={2}>
    <div style={{ width: '100px', backgroundColor: 'pink' }} />
    <div style={{ width: '100px', backgroundColor: 'darkmagenta' }} />
    <div style={{ width: '100px', backgroundColor: 'pink' }} />
    <div style={{ width: '100px', backgroundColor: 'darkmagenta' }} />
 </Flex>
```

- Wrap items wrap-reverse

```ts
import { Flex } from '@grafana/ui'

<header>
    <h1>Using Flex component to align-items with wrap-reverse property</h1>
</header>
<Flex flexFlow="row wrap-reverse" alignItems="center" justifyContent="center" gap={4}>
    <div style={{ width: '100px', height: '100px', backgroundColor: 'pink' }} />
    <div style={{ width: '100px', height: '100px', backgroundColor: 'darkmagenta' }} />
    <div style={{ width: '100px', height: '100px', backgroundColor: 'pink' }} />
    <div style={{ width: '100px', height: '100px', backgroundColor: 'darkmagenta' }} />
 </Flex>
```

- JustifyContent flex-start

```ts
import { Flex } from '@grafana/ui'

<header>
    <h1>Using Flex component to align-items with justify-content property</h1>
```

- Direction column

```ts

    <h1>Using Flex component to align-items with direction column</h1>

```

### Dos

    * to display the direction of the elements
    * to set the elements to wrap
    * to align items (center, vertically or horizontally)

### Don'ts

    * for complex grid lay outs with various rows and columns
    * bidirectional lay outs
    * complex nesting
    * equal height columns
