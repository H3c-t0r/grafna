How to use:

  • To create a main header create a blank dashboard with the tag L1.

  • To put a dashboard under a main header you simply give it a tag with the exact name of the dashboard that you gave with the tag L1.

  • To create a sub menu off of an L1 tagged dashboard, you create a blank dashboard and assign two tags. One L2 tag to indicate this is a sub menu and the other tag is the name of a dashboard that has an L1 tag where the sub menu should appear.

  • To put a dashboard under a sub menu you simply give it a tag with the exact name of the dashboard that you gave with the tag L2.
