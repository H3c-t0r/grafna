# ArcGIS

A layer from an ESRI ArcGIS MapServer.

## Options

- **Server Instance** to select the map type.
  - World Street Map
  - World Imagery
  - World Physical
  - Topographic
  - USA Topographic
  - World Ocean
  - Custom MapServer (see [XYZ]({{< relref "xyz/" >}}) for formatting)
    - URL template
    - Attribution
- **Opacity** from 0 (transparent) to 1 (opaque)
